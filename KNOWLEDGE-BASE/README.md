# KNOWLEDGE-BASE

This folder contains your personal data that drives the job-apply skill.
**Do not commit these files to any public repo.**

## Files to create locally

### `cv-general.md`
Your base CV in Danish. The job-apply skill uses this as the foundation
for every tailored application. Include: name, contact info, work history,
skills, education, languages.

### `cv-general-en.md`
Same CV in English for international roles.

### `employment-history.md`
Detailed work history with specific achievements, project names, and
technologies. More detail than the CV — the skill draws on this for
proof points in cover letters.

### `faglig-praesentation.md`
A 2–3 paragraph professional presentation in your own voice.
Used as inspiration for the About section of cover letters.

### `projects-and-expertise.md`
Key projects you've worked on: what it was, your role, technologies, outcomes.
The skill uses this for the match analysis table.

## Privacy

These files stay on your machine. The `.gitignore` in this repo excludes
all `KNOWLEDGE-BASE/*.md` files except this README.
